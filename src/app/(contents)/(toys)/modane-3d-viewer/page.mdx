# もだねちゃん 3D ビューワー

[ねんどろいど もだねちゃん](https://kems-clew.net/assets/works/large/20211016_modane_nendoro.jpg) の [Three.js](https://threejs.org/) 製ビューワーです。

- 回転: 左クリックでドラッグ / スワイプ
- 移動: 右クリックでドラッグ / 二本指ドラッグ
- ズーム: マウスホイール / ピンチイン・アウト

<ModaneThree />

※ 動作が重い場合は、ブラウザの次の設定がオンになっているか確認してください。

- Microsoft Edge: `設定` > `システム` > `使用可能な場合はグラフィックス アクセラレータを使用する`
- Google Chrome: `設定` > `システム` > `グラフィック アクセラレーションが使用可能な場合は使用する`
