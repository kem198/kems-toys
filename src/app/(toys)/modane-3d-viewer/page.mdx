# もだねちゃん 3D ビューワー

[ねんどろいど もだねちゃん](https://clew.kem198.net/assets/works/large/20211016_modane_nendoro.jpg) を 3D スキャンしました。

- 回転: 左クリックでドラッグ / スワイプ
- 移動: 右クリックでドラッグ / 二本指スワイプ
- ズーム: マウスホイール / ピンチイン・アウト

<Modane3dViewer />

※ 動作が重い場合は、ブラウザの次の設定がオンになっているか確認してください。

- Microsoft Edge: `設定` > `システム` > `使用可能な場合はグラフィックス アクセラレータを使用する`
- Google Chrome: `設定` > `システム` > `グラフィック アクセラレーションが使用可能な場合は使用する`
