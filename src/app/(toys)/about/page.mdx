# About

## このサイトは何？

[KeM's Toys](https://toys.kem198.net) (以下「当サイト」) は、管理人である KeM198 が制作・公開しているウェブサイトです。ウェブフロントエンド系技術の練習用として制作されました。

当サイトのソースコードは GitHub にて公開しています。

- [kenkenpa198/kems-toys](https://github.com/kenkenpa198/kems-toys)
