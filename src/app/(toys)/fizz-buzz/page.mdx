# Fizz Buzz

与えられた自然数 n について、

- 3 で割り切れる場合は `Fizz!` を返す。
- 5 で割り切れる場合は `Buzz!` を返す。
- 両方で割り切れる場合は `Fizz Buzz!!` を返す。
